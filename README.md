# Local Env

## Requirements

- Python 3.11

## How to install and run this project

-> clone this repo
-> cd into src folder
-> Create a venv 'py -m venv venv'
-> Activate venv 'venv\scripts\activate'
-> Install dependencies 'pip install -r requirements.txt'
